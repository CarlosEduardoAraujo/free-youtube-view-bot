Welcome to **YT Brave Bot GUI**! This is a complete tool for those looking to automate YouTube views with full privacy.

### Highlights

- ✨ **Minimalist UI:** Premium Dark Mode design.  
- 🔒 **Privacy First:** Forced browsing through Tor.  
- ⚙️ **Configurable:** Automatically saves your preferences.  
- 📦 **Portable:** Available as a standalone executable.

This project was created to provide a **simple, modern, and privacy-focused solution** for automating YouTube views using **Brave Browser** with **Tor network routing**.

The goal of this tool is to combine **ease of use**, **privacy**, and **automation** in a clean and lightweight interface.

---

## ✨ Key Features

- 🤖 **Automated YouTube View Sessions**
- 🌐 **Brave Browser Integration**
- 🔒 **Tor Network Routing for Privacy**
- 🖥️ **Minimalist Dark Mode Interface**
- ⚙️ **Automatic Configuration Persistence**
- 📦 **Portable Standalone Executable**
- 🔁 **Multi-Instance Control**

## 🖥️ User Interface

The interface was designed to be **minimalist and intuitive**, allowing users to quickly configure and start automation sessions without complex setup.

Everything is accessible from a **single window**, focusing on simplicity and productivity.
